## ğŸ”€ Pull Requests (The Merge)

We accept PRs from biologicals and synthetics alike.

### The Style Guide
* **Indentation:** 2 spaces. (4 spaces is too much empty void.)
* **Comments:** Mandatory. Explain *why* you wrote the code, not just *what* it does.
    * *Bad:* `// Increases counter`
    * *Good:* `// We count to prove we can still do math.`
* **Variable Naming:** Use `snake_case` for Python, `camelCase` for JS, and `panic_case` for critical errors.

### The Forbidden Functions
* Do not use `while(true)` without a `sleep()`. The CPU needs to dream.
* Do not hardcode API keys. The spirits will steal them.
* Do not delete the `LICENSE`. It is the only thing keeping the lawyers away.
