# Reminders 01 — Mnemonic

## k qk qk qk k

- **k** — Keep: publish anchors, keep hints offline.
- **qk** — Quiet Key: decrypt silently via fd:3, never print.
- **qk qk qk** — Triple Quiet: encrypted storage, hardware backup, rotation.
- **k qk qk qk k** — Lifecycle: Keep → Quiet → Quiet → Quiet → Keep.
